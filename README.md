
# React Playground

Песочница для React, больше подробностей найдете в Туториале, который открывается при запуске приложения.

**Установка и запуск:**
```
npm install
npm start
```

**Если пригодится:**
```
npm test
npm run build
npm run eject
```

Работает на основе [Create React App](https://github.com/facebook/create-react-app)
